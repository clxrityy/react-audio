# ShufflePlayer

The `<ShufflePlayer />` component is built off of the [Player](/docs/components/player) component, adding the ability to play multiple audio files.

<br />

```tsx
"use client";
import { ShufflePlayer } from "@clxrityy/react-audio";

export default function Example() {
  return (
    <ShufflePlayer
      srcs={[
        "/audio/strings.wav",
        "/audio/drums_2.wav",
        "/audio/drums.wav",
      ]}
    />
  );
}
```
