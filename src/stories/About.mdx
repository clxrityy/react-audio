      <h4 className="sb-section-item-heading">
        Props
      </h4>
      <p className="sb-section-item-paragraph">
        Every component has its own set of props, however the following are common to all components:

        - `track`/`tracks` - The source of the audio file(s).
        - `autoplay` - Whether the audio should start playing automatically.
        - `btnStyleProps` - The style of the button(s) within the component.

        <div className="snippet">
          ```ts
          /**
           * The track object.
           * @param title - The title of the track.
           * @param src - The source of the track.
           * @param thumbnail - The thumbnail of the track.
           * @param author - The author of the track.
           */
          type Track = {
            title?: string;
            src: string;
            thumbnail?: string;
            author?: {
              name: string;
              url?: string;
            }
          }

          /**
           * The button props.
           * @param color - The color of the button.
           * @param theme - The theme of the button.
           * @param size - The size of the button.
           */
          interface Buttonrops extends ComponentPropsWithoutRef<"button"> {
            color?: string;
            theme?: 'light' | 'dark';
            size?: "sm" | "md" | "lg";
          }

          /**
           * PROPS
           */
          interface Props {
            track: Track;
            autoplay?: boolean;
            btnStyleProps?: ButtonProps;
          }

          /**
           * Ex. Usage
           */
          interface AudioPlayerProps extends Props {
            // ...
          }
          ```
          
        </div>

        Along with those base props, components with a waveform/visualizer ([Waveform](https://clxrityy.github.io/react-audio/?path=/docs/components-waveform--docs) / [AudioInputVisualizer](https://clxrityy.github.io/react-audio/?path=/docs/components-waveform--docs)) also have additional props such as [fftsize](https://developer.mozilla.org/en-US/docs/Web/API/AnalyserNode/fftSize):
